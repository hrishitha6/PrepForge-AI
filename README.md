# 🚀 PrepForge AI

## Overview

PrepForge AI is an internship readiness platform that analyzes a candidate's resume against a target job description and provides personalized preparation insights.

The application extracts skills from resumes, compares them with job requirements, identifies skill gaps, calculates a readiness score, and generates AI-powered recommendations for interview preparation and learning.

---

## Features

### Resume Analysis
- PDF Resume Upload
- Resume Text Extraction
- Skill Detection

### Skill Matching Engine
- Matching Skills Identification
- Missing Skills Detection
- Skill Gap Analysis

### Internship Readiness Scoring
- Readiness Score Calculation
- Technical Match Evaluation

### AI-Powered Recommendations

- Interview Preparation Questions
- Learning Resources
- Project Suggestions

### Visualization

- Skill Comparison Charts
